关于 Mirror酱

### 什么是 Mirror酱

- Mirror酱 是一个应用分发平台，国内、海外均能保证优良的连通性和下载速度
- CDK 是用户付费凭证，加速下载服务是付费的
- 即使不付费也很有用——可以及时获取新版本通知和版本差异信息，之后使用 GitHub 源一样可以更新

### 现行更新方式

🌏 **GitHub**

- **更新下载**：从 GitHub 拉取更新包，国内需要解决连通性问题
- **版本检测**：由 Mirror酱 提供免费的更新检测

🔑 **Mirror酱**（推荐）

填写 [CDK](https://mirrorchyan.com/zh/projects?rid=MBCCtools) 后，支持软件与资源的快速更新，通过高速 CDN 下载，更稳定快速。

> 年卡只要 30 元，所有 Mirror酱 支持的项目均可使用同一 CDK，告别 GitHub 下载龟速！

📦 **手动更新**

下载 [资源包](https://github.com/quietlysnow/MBCCtools/releases) 后，直接覆盖现有的文件夹，如遇问题请尝试重新下载。
